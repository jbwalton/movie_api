The Magical List of URL endpoints

  • Return a list of ALL moveies to the user
  • Return data (description, genre, director, image URL, whether it's featured or not)
    about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., "Thriller")
  • Return data about a director (bio, birth year, death year) by name
  • Allow news users to register
  • Allow users to add a movie to their list of favourites (showing only a text that a movie
    has been added) - more on this later
  • Allow users to remove a movie from their list of favourites (showing only a text that a movie
    has been removed) - more on this later
  • Allow existing users to deregister (showing only a text that a user email has been
    removed) - more on this later
